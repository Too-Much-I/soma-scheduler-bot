소마 멘토링/특강 노션 자동 등록기 (Soma Scheduler Bot)
소마 멘토링 및 특강 신청 시, 일일이 팀 노션 페이지에 등록해야 하는 번거로움을 해결하기 위한 자동화 도구입니다.

🛠 사전 준비 사항 (팀원 필수)
확장 프로그램을 사용하기 전, 팀원들은 아래 과정을 반드시 따라야 합니다.

1. 확장 프로그램 설치
이 레포지토리를 [Code] -> [Download ZIP] 버튼을 눌러 다운로드하고 압축을 해제합니다.

크롬 브라우저에서 chrome://extensions/ 페이지로 이동합니다.

우측 상단의 [개발자 모드]를 켭니다.

좌측 상단의 [압축해제된 확장 프로그램을 로드합니다]를 클릭하여 압축 푼 폴더(frontend 폴더)를 선택합니다.

2. 네트워크 환경 체크 (중요!)
네트워크: 특정 학교/회사 망에서는 통신을 차단할 수 있습니다. 정상 작동하지 않는다면 개인 핫스팟이나 다른 네트워크에서 시도해 주세요.

서버 상태: 프로그램이 정상적으로 작동하려면 팀 공용 서버(Replit)가 켜져 있어야 합니다. (서버가 꺼져 있다면 성환 님에게 서버를 켜달라고 요청하세요!)

3. 사용 방법
[소마 마이페이지 - 멘토링/특강 신청 페이지]로 이동합니다.

원하는 항목의 [신청하기] 버튼을 클릭합니다.

브라우저 상단에 "✅ [자동화] 멘토링 일정이 노션 달력에 등록되었습니다!"라는 알림창이 뜨면 성공입니다.

팀 노션 페이지를 새로고침하여 일정이 추가되었는지 확인합니다.

💡 Troubleshooting (안 될 때 확인하세요!)
증상확인 사항반응이 없음개발자 도구(F12) 콘솔에서 "👀 소마 확장 프로그램 content.js 정상 주입 완료!"가 찍히는지 확인하세요.자동 전송 실패 (Failed to fetch)서버가 켜져 있는지 확인하고, 네트워크가 외부 서버와 통신이 가능한지(방화벽 확인) 체크하세요.노션에 안 보임노션 페이지 상단의 **[필터]**를 '없음'으로 설정했는지, 데이터베이스 속성(이름, 일시 등)이 정확한지 확인하세요.빨간 에러 로그광고 차단기(AdBlock)로 인한 흔한 에러일 수 있으니, 알림창이 뜬다면 무시해도 됩니다.
